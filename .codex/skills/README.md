# Skills Index

- `openspec-apply-change`: Implement tasks from an OpenSpec change.
- `openspec-archive-change`: Archive a completed OpenSpec change.
- `openspec-explore`: Explore ideas and requirements without implementing.
- `openspec-propose`: Create a new OpenSpec change and artifacts in one step.
- `tauri-v2-app-skill`: Implement and validate Tauri v2 app work with explicit frontend/Rust boundaries and run/build/package checks.
